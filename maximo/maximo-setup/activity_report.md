## How this report is consumed

- **Setup interview** reads this report in Batch 0 Q2 (Heatmap Review). The customer confirms or overrides each verdict; the result is captured as `industry_usage.modules_in_scope` in `answers.json`.
- **`apply_uc_comments.py --scope active`** (Phase 3 Optional) consumes this report's verdicts via `draft_profile.json["module_activity"]`. DORMANT modules' tables are excluded from the comments preview by default.
- **`generate_glossary.py`** carries the active modules into the rendered glossary as a "Modules in scope" section, plus caveats from low cross-table population values.
