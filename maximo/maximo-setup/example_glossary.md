## Asset / location hierarchy

- **Region** → `LOCATIONS` level 1 (top-level parents like `REG-NORTH`, `REG-SOUTH`)
- **Station** → `LOCATIONS` level 2 (compressor stations, pump stations)
- **Unit** → `LOCATIONS` level 3 (named units within a station)
- **Equipment** → leaf-level `ASSET` records

Traverse via `LOCHIERARCHY` for queries that need full path resolution.

## Tribal knowledge

- "In service" colloquially means `WORKORDER.STATUS = 'INPRG'`, not the formal `ASSET.STATUS = 'OPERATING'`. Disambiguate.
- Region 5 was renamed to "North" in 2024 but the `LOCATIONS` records still say `REG5`. Both terms map to the same site list.
- "Pipeline integrity inspections" are tracked as work orders with `WORKTYPE = 'API570'` AND `WO_REGULATORY_FLAG = 'Y'`. Other types of inspection work do not have `WO_REGULATORY_FLAG = 'Y'` set. (Fictional example — the actual flag-column name varies per customer.)
- HSE module is sparsely populated for the first half of 2022 — known data quality gap from a migration. Filter HSE analytics to >= 2022-07-01.
