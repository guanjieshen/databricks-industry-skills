# Northstar Energy — WellView Glossary

Generated by `wellview-setup` on 2026-06-04. Edit when schema, codes, or units change.

## Catalog / schema
- WellView Silver: `wellview.silver` (replicated via Peloton ETL on Snowflake → Databricks)

## Spine tables (canonical → physical)
| Canonical | Physical table | Grain | Parent edge |
|---|---|---|---|
| well | `WVWELLHEADER` | 1 / well | root (`IDREC = IDWELL`) |
| job | `WVJOB` | 1 / job | `IDWELL` → well |
| daily report | `WVJOBREPORT` | 1 / day / job | `IDRECPARENT` → `WVJOB.IDREC` |
| time log | `WVJOBREPORTOP` | 1 / activity | `IDRECPARENT` → `WVJOBREPORT.IDREC` |
| cost | `WVJOBREPORTCOST` | 1 / cost line | `IDRECPARENT` → **`WVJOBREPORT.IDREC` (confirmed: report-level)** |
| AFE | `WVAFE` | 1 / AFE | `IDRECPARENT` → `WVJOB.IDREC` |

## Master units (confirmed in interview)
| Column | Table | Master unit | Notes |
|---|---|---|---|
| `DEPTHMD`, `DEPTHTVD` | `WVJOBREPORT` | **feet** | confirmed; UI also shows feet |
| `DEPTHSTART`, `DEPTHEND` | `WVJOBREPORTOP` | **feet** | per-activity footage = end − start |
| `HRS` | `WVJOBREPORTOP` | **hours** | reconcile to ~24/report-day |
| `AMOUNT` | `WVJOBREPORTCOST` | **USD** | single-currency operator |
| `MUDWEIGHT` | `WVDAILYMUD` | **ppg** | — |

## LV code decodes
| Coded column | LV table | Code→label | NPT class? |
|---|---|---|---|
| `CODEOP` (operation) | `LVWVCODEOP` | `Code` → `Description` | — |
| `CODENPT` (NPT reason) | `LVWVCODENPT` | `Code` → `Description` | non-null ⇒ NPT |
| `CODECOST` | `LVWVCODECOST` | `Code` → `Description` | `Category`: Tangible/Intangible |
| `JOBTYPE` | `LVWVTYPEJOB` | `Code` → `Description` | Drill/Compl/Workover/P&A |

NPT rule (Northstar): an activity is NPT when `WVJOBREPORTOP.PRODUCTIVE = 0` **or**
`CODENPT IS NOT NULL`.

## Calc-vs-stored
| Metric | Stored column? | If not, recompute via |
|---|---|---|
| days from spud | **stored** `DAYSFROMSPUD` on `WVJOBREPORT` | — |
| cumulative cost | **NOT stored** | window sum of `WVJOBREPORTCOST.AMOUNT` by job, ordered by date |
| ROP | **NOT stored** | footage ÷ on-bottom hours UDF (`wellview_rop`) |

## Business jargon
- **"The Permian wells"** → `WVWELLHEADER.FIELD IN ('MIDLAND','DELAWARE')`
- **"Open / active job"** → `WVJOB.STATUS IN ('ACTIVE','INPROGRESS')`
- Rigs: "Rig 7" → `WVJOBRIG.RIGNAME = 'NS-007'`
- Cost rollup: "tangibles" → `LVWVCODECOST.Category = 'Tangible'`

- drilling ✅ · completion ✅ · workover ✅ · integrity ❌ (not adopted)
